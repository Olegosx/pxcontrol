# Модель данных

> Статус: 🟢 актуально · Обновлено: 2026-07-16 · Владелец: olegosx

## Назначение документа

Описать сущности приложения, их поля и связи. Соответствует коду
`src/pxcontrol/engine/db/models.py` (миграции `e5cc5457ed7a` … `c8f1d29e4a35`).

## Принципы

- СУБД — **SQLite** (локальный файл), схема версионируется **Alembic**;
  миграции применяются при старте (`Database.init()`).
- Поля, помеченные 🔐, шифруются прозрачно (тип `EncryptedStr`, ключ —
  в системном хранилище ОС; [ADR-0009](../02-architecture/decisions/0009-config-secrets-in-db.md)).
- Медиа-файлы в БД не хранятся — только пути; сами файлы на диске.
- У всех таблиц, кроме `app_settings`, `channel_settings`
  и `caption_template_fields`, есть `created_at`/`updated_at`.

## Сущности (текущая схема)

### `app_settings` — настройки приложения (ADR-0013)
| Поле | Тип | Назначение |
|---|---|---|
| `name` | str, PK | имя настройки (из реестра ключей) |
| `value` | JSON | значение (типизированный скаляр) |

### `channel_settings` — настройки канала (ADR-0013)
| Поле | Тип | Назначение |
|---|---|---|
| `channel_id` | FK→channels, PK | канал (каскадное удаление) |
| `name` | str, PK | имя настройки (из реестра ключей) |
| `value` | JSON | значение |

Хранятся только отличия от умолчаний; состав, типы и умолчания задаёт
реестр ключей (`engine/services/settings.py`). Текущие ключи канала:
`enabled` (канал активен: публикация и расписание, миграция `c8f1d29e4a35`),
`default_video_preset` (пресет обработки по умолчанию; целостность ссылки
держит сервис — вариант «а» ADR-0013).

### `bots` — Telegram-боты (несколько; канал ссылается на своего)
| Поле | Тип | Назначение |
|---|---|---|
| `label` | str | человекочитаемое имя |
| `token` 🔐 | str | токен Bot API |
| `username` | str? | @имя бота, полученное при проверке токена (getMe); миграция `b41c7a9d2e10` |

### `tg_accounts` — userbot-аккаунты MTProto (ADR-0007)
| Поле | Тип | Назначение |
|---|---|---|
| `label`, `phone` | str | имя, номер (справочно) |
| `api_id` | int | ID приложения с my.telegram.org |
| `api_hash` 🔐 | str | hash приложения |
| `session` 🔐 | str? | строка сессии (секрет уровня пароля), после входа |

### `ai_credentials` — ключи провайдеров ИИ
| Поле | Тип | Назначение |
|---|---|---|
| `provider` | str | провайдер (по умолчанию `anthropic`) |
| `label` | str | имя ключа |
| `api_key` 🔐 | str | ключ API |

### `video_presets` — шаблоны обработки видео (референс makeVideo)
`name`, `trim_start`/`trim_end` (обрезка с краёв, сек; 0 — не резать;
остальные параметры — от обрезанной версии; миграция `d2b8e51c7f94`),
`fade_in`/`fade_out` (затухание на краях итога, сек; 0 — без эффекта;
миграция `b9e2c74a51d3`),
`watermark_path?`, `wm_corner/margin/opacity/scale`,
`wm_start_offset?/wm_end_offset?` (окно показа вотермарка отступами
от начала/до конца, сек; миграция `f2b8d61c3a97`), `wm_fade`
(плавность переходов, сек; миграция `a3c5e97d21b4`), `intro`,
`intro_source`, `intro_hold`, `xfade`, `cover`, `no_audio`,
`video_bitrate_kbps?` (NULL — «как в оригинале»; миграция
`c7d2f4a91b33`), `meta_comment?` (тег comment в метаданные файла:
«ссылка на канал — описание»; миграция `e9c2a71f4d36`).
Переиспользуются между каналами. Значения по умолчанию полей заданы
в коде в одном месте — `PresetFields` (сервис видео); запись всегда
создаётся со всеми полями.

### `channels` — подключённые каналы
| Поле | Тип | Назначение |
|---|---|---|
| `title` | str | название |
| `tg_chat_id` | str, unique | идентификатор канала в Telegram |
| `username` | str? | @имя, если есть |
| `userbot_admin` | bool | userbot — админ канала (проверено при подключении; миграция `f4e8b3c92a17`) |
| `bot_id` | FK→bots? | бот-админ (NULL — канал подключён через userbot) |

### Подписи к постам (миграция `d9f3a6e42b71`, [captions.md](../03-modules/captions.md))

- **`caption_fields`** — пул полей канала: `channel_id` FK, `name`,
  `hashtag` (оформлять значения решётками), `multiple` (несколько значений).
- **`caption_values`** — словарь значений поля: `field_id` FK, `value`.
  Общий для всех шаблонов канала.
- **`caption_templates`** — именованные шаблоны: `channel_id` FK, `name`,
  `last_used_at?` (для предвыбора в диалоге), `filename_pattern?` —
  шаблон имени файла при отправке (миграция `e7a1c94f5d28`).
- **`caption_template_fields`** — состав шаблона: `template_id` FK,
  `field_id` FK, `position`, `enabled`. Без полей времени.

## Отложено сознательно (YAGNI)

`media`, `sources`, `queue_items`, `generation_jobs` — добавляются
миграциями вместе с соответствующими функциями (решение от 03.07.2026).

Таблицы `posts` **нет намеренно** (ADR-0010): источник истины по постам —
сам канал; отложенные читаются из Telegram. В БД — только конфигурация.
