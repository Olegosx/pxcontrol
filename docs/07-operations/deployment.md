# Развёртывание

> Статус: 🟡 черновик-заглушка · Обновлено: 2026-06-30 · Владелец: olegosx

## Назначение документа

Описать упаковку и установку приложения. Это **десктоп**
([ADR-0006](../02-architecture/decisions/0006-desktop-pyside6.md)), поэтому
«развёртывание» — это сборка пакета и установка на машину пользователя,
а не серверный хостинг.

## Что распространяем (план)

- [ ] Единый пакет приложения (движок + интерфейс PySide6).
- [ ] Локальная БД SQLite создаётся при первом запуске.
- [ ] Зависимость: `ffmpeg`/`ffprobe` в системе (для обработки видео).

## Открытые вопросы

- [ ] Способ упаковки: PyInstaller / Briefcase / иное.
- [ ] Платформы: Linux / Windows / macOS.
- [ ] Поставлять `ffmpeg` в комплекте или требовать установленный в системе?
- [ ] Автообновление приложения.
- [ ] Резервное копирование локальной БД и настроек.
