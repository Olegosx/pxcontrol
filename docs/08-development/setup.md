# Настройка окружения разработчика

> Статус: 🟡 черновик-заглушка · Обновлено: 2026-06-30 · Владелец: olegosx

## Назначение документа

Пошаговая инструкция, как поднять проект локально для разработки.
Заполняется, когда появится код и зафиксируется стек.

## Шаги (план)

- [ ] Версия Python и менеджер окружений — **uv**.
- [ ] Установка зависимостей.
- [ ] Установка `ffmpeg`/`ffprobe` в системе (для обработки видео).
- [ ] Копирование `.env.example` → `.env` и заполнение значений.
- [ ] БД SQLite — локальный файл, создаётся автоматически.
- [ ] Запуск десктоп-приложения (движок + интерфейс PySide6).
- [ ] Проверка: окно открывается, тестовый пост уходит в тестовый канал.

## Требования

- [ ] Тестовый Telegram-бот и тестовый канал для разработки.
- [ ] Ключ провайдера ИИ (для функций генерации).
