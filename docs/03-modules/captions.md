# Модуль «Подписи к постам»

> Статус: 🟢 реализовано · Обновлено: 2026-07-14 · Владелец: olegosx

## Назначение

Быстрая сборка подписей вида «жирное название + строки "Поле: #значения"»
на странице «Публикация». Цель — подпись к видео за несколько кликов.

## Модель

- **Поле канала** (`caption_fields`) — имя («Genre», «Year»…), оформление
  значений (решётки/текст), множественность. У поля — **словарь значений**
  (`caption_values`), общий для всех шаблонов канала: значение, добавленное
  при сборке по одному шаблону, сразу видно в остальных.
- **Шаблон** (`caption_templates` + `caption_template_fields`) — именованный
  упорядоченный набор полей канала. Шаблонов у канала сколько угодно;
  в диалоге предвыбирается последний использованный (`last_used_at`).

## Поведение

- `CaptionsService` — CRUD полей/шаблонов, `record_usage` (автопополнение
  словарей использованными значениями, без дублей по регистру).
- Чистые функции: `build_caption` (название `**жирным**` — Markdown,
  Telethon применяет его по умолчанию; пустые поля пропускаются),
  `hashtag` («Tomb Raider» → `#TombRaider`), `title_from_filename`
  (имя файла без суффикса конвейера `_<пресет>_<штамп>`).
- Интерфейс: на «Публикации» — «Собрать подпись…» (диалог: шаблон,
  название из имени файла, поля со словарями; ненужные поля отключаются
  галочками) и «Поля подписи…» (настройка пула полей и шаблонов;
  порядок полей шаблона — перетаскиванием). Результат вставляется
  в поле подписи и правится руками.

## Переименование файла при отправке

У шаблона подписи есть необязательный **шаблон имени файла**
(`filename_pattern`), например `{Author}, {title} ({Genre}) {quality}
(@{channel})`. Плейсхолдеры: `{title}` — название; `{ИмяПоля}` — значения
поля через запятую (без решёток); `{quality}` — меньшая сторона кадра
видео (ffprobe); `{channel}` — @имя канала; неизвестные остаются как есть.

После сборки подписи движок вычисляет имя (`render_filename`: подстановка,
очистка недопустимых символов, расширение исходника), на странице
появляется строка «Переименовать при отправке» — имя правится руками,
галочка отключает. При отправке `PostsService` переименовывает файл
**и его кадр-превью** на диске (`PostDraft.rename_to`), занятое имя
или путь в имени — понятная ошибка до отправки.
