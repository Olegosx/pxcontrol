# Модуль «Генерация ИИ»

> Статус: 🟡 черновик-заглушка · Обновлено: 2026-06-30 · Владелец: olegosx

## Назначение

Создавать контент нейросетью: тексты постов по теме и стилю канала,
рерайт материалов из источников, при необходимости — изображения.

## Ответственность (план)

- [ ] Генерация текста по заданию (тема, тон, длина, стиль канала).
- [ ] Рерайт исходного материала из [источников](content-sources.md).
- [ ] (Опционально) генерация/подбор изображений.
- [ ] Хранение черновиков для правки человеком перед публикацией.

## Открытые вопросы

- [ ] Провайдер ИИ: Claude API (рекомендуется по стеку) / иной. Зафиксировать в tech-stack.
- [ ] Где хранить «стиль канала» (промпт-шаблон, примеры постов)?
- [ ] Контроль качества и фактчекинг сгенерированного.
- [ ] Учёт стоимости вызовов ИИ (бюджет/лимиты).

## Безопасность

- [x] Ключ провайдера ИИ — никогда в коде: хранится в БД в зашифрованном
      виде (таблица `ai_credentials`, [ADR-0009](../02-architecture/decisions/0009-config-secrets-in-db.md)),
      управляется на странице «Настройки → Аккаунты»
      (см. [configuration.md](../06-config/configuration.md)).
