# Модуль «Источники контента»

> Статус: 🟡 черновик-заглушка · Обновлено: 2026-06-30 · Владелец: olegosx

## Назначение

Регулярно собирать материалы из внешних источников (другие каналы, сайты,
RSS-ленты), нормализовать их и передавать дальше — в очередь модерации
и/или на рерайт нейросетью.

## Ответственность (план)

- [ ] Подключение источников (URL сайта, адрес RSS, ссылка на канал).
- [ ] Регулярная вычитка по таймеру (в фоновом обработчике).
- [ ] Нормализация в единый внутренний формат «заготовки контента».
- [ ] Дедупликация — не брать одно и то же повторно.
- [ ] Передача в [очередь модерации](moderation-queue.md).

## Telegram-каналы как источники

Чтение чужих каналов обеспечивает **MTProto** (`Telethon`, отдельный аккаунт),
см. [ADR-0007](../02-architecture/decisions/0007-hybrid-botapi-mtproto.md) —
это и было главной причиной подключить MTProto. Bot API такого не умеет.
Чтение ведём аккуратно (лимиты частоты), чтобы не подставлять аккаунт под бан.
