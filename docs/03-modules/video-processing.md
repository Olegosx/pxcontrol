# Модуль «Подготовка видео»

> Статус: 🟢 реализовано (подготовка) · Обновлено: 2026-07-12 · Владелец: olegosx

## Назначение

Готовит видеофайл к публикации в Телеграме: FullHD-вписывание, вотермарк,
заставка для красивого превью, обложка. Порт наработок из референса
`makeVideo` (поведение — по его `SPEC.md`).

## Архитектурная граница (важно)

**Подготовка отделена от публикации.** Контракт — путь к файлу:

- `engine/video/` — **чистый модуль**: файл + `ProcessingOptions` →
  обработанный файл. Не знает про Telegram, каналы и БД. Синхронный
  (как и его тесты); async-граница — в сервисе (`asyncio.to_thread`).
- `services/video.py` — `VideoService`: пресеты в БД (`video_presets`),
  маппинг пресета в `ProcessingOptions`, пути (`media/processed/`
  в папке приложения), человеческие ошибки.
- Публикация готового файла — зона `PostsService` (слайс B, следующий шаг).

## Состав чистого модуля

| Файл | Что делает |
|---|---|
| `probe.py` | метаданные через ffprobe (размеры, длительность, fps, звук) |
| `filtergraph.py` | сборка `-filter_complex`: FullHD, вотермарк с окном показа, заставка hold+xfade — чистые функции |
| `frames.py` | кадр заставки: random-middle / first / time / frame / image |
| `pipeline.py` | сборка и запуск команды ffmpeg, вшивание обложки (`attached_pic`) |

## Пресеты

Таблица `video_presets` (все параметры makeVideo). Управляются со страницы
«Видео»: создание/правка/удаление; при обработке выбираются из списка.
Привязка «пресет по умолчанию для канала» (`channels.video_preset_id`) —
заиграет в слайсе B.

## Интерфейс

Страница «Видео»: пресеты (карточки + диалог со всеми параметрами) и блок
подготовки: исходник → пресет → «Обработать» → **полоса прогресса
с процентами** (ffmpeg отчитывается через `-progress pipe:1`; кодирование —
в фоновом потоке, окно живо) → карточка результата («Открыть»,
«Показать в папке»).

## Ограничения текущей версии

- Просмотр результата — системным плеером; встроенный (QtMultimedia) — позже.
- Окно показа вотермарка (`wm_start`/`wm_end`) в диалог не вынесено
  (поля в БД есть).
