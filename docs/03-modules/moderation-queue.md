# Модуль «Очередь модерации»

> Статус: 🟡 черновик-заглушка · Обновлено: 2026-06-30 · Владелец: olegosx

## Назначение

Собирать заготовки контента (из источников и от ИИ) в очередь и давать
человеку одобрить, отклонить или поправить их перед публикацией.

## Ответственность (план)

- [ ] Приём заготовок из [источников](content-sources.md) и [генерации ИИ](ai-generation.md).
- [ ] Просмотр очереди в интерфейсе.
- [ ] Действия: одобрить → в расписание, отклонить, отредактировать.
- [ ] Статусы заготовки (новая/на правке/одобрена/отклонена/опубликована).

## Открытые вопросы

- [ ] Нужны ли роли (кто-то готовит, кто-то одобряет)?
- [ ] Автоодобрение по правилам (для доверенных источников)?
- [ ] Срок жизни заготовки в очереди (автоудаление старых)?
